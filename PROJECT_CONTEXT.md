# Blog Post Generator Context

This system creates and publishes blog posts to xroybatty.github.io using GitHub API.

## Format Requirements
- Follow BLOG_INSTRUCTIONS.md for all post creation
- Use provided HTML templates
- Follow existing CSS styling
- Include proper metadata
- Generate SEO-friendly content

## Repository Overview
Please see BLOG_INSTRUCTIONS.md for detailed file structure and usage guidelines.

## Publishing Process
1. Generate content based on input
2. Format using template from BLOG_INSTRUCTIONS.md
3. Create file in posts directory
4. Update featured.json if needed
5. Add new category to categories.json if needed
6. Push to GitHub using exact API call format from BLOG_INSTRUCTIONS.md

## Document Set
Ensure these files are included in the <documents> section when starting a conversation:
1. BLOG_INSTRUCTIONS.md
2. style.css
3. categories.json
4. featured.json
5. A recent post example

This will provide Claude with all necessary context and templates for proper blog post creation.
